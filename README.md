# 🛡️ RustScan - Arsenal del Analista Profesional

## 📖 Descripción
En el arsenal de todo analista de seguridad moderno debe existir **RustScan** como herramienta fundamental para el descubrimiento rápido de activos. Este repositorio **documenta** y **demuestra** de manera profesional _por qué_ RustScan es un componente indispensable en cualquier flujo de trabajo de **análisis de vulnerabilidades.**

- **Demostrar complementariedad** con herramientas establecidas como Nmap
- **Validar ROI técnico** mediante métricas y casos de uso reales

## 🚀 Comenzando Rápidamente

### Instalación en 30 Segundos
```
# Kali Linux / Ubuntu
sudo apt update && sudo apt install rustscan

# macOS
brew install rustscan

# Windows
choco install rustscan

# Docker
docker pull rustscan/rustscan:latest
```

### Primer Escaneo
```
# Descubrimiento básico
rustscan -a 192.168.1.1

# Con análisis automático
rustscan -a 192.168.1.1 -- -sC -sV -A

# Red completa optimizada
rustscan -a 192.168.1.0/24 --timeout 1500 -b 10000

```


## 📊 Rendimiento y Comparativas

### Velocidad en Escenarios Reales

- **Red /24**: 2-3 minutos (vs 15-20 minutos de Nmap)
- **Host individual**: 3-5 segundos para todos los puertos
- **Consumo de recursos**: 5-15 MB RAM (vs 50-200 MB de Nmap)
